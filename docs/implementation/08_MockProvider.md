# MockProvider Specification

Version: 1.0

Status: Approved

Sprint: Sprint-01 Runtime Foundation

Parent Interface

BaseProvider

---

# Purpose

MockProvider is the first concrete implementation of BaseProvider.

It allows AgentForge Runtime development without requiring:

* OpenAI
* Anthropic
* Gemini
* Ollama
* OpenRouter

or any external API.
